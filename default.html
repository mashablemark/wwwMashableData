Sharing data and visualizatons across the web

Technology to simplify data sharing and visualization, across a website or across the Internet

MashableData.com simplifies data sharing, mashups and visualizations, across a web site or across the Internet. Using MashableData.com plug-ins to common data visualization tools allow a user to save data series to their browser's localStorage (now supported by all major browsers). Unlike cookies, the data stays local and is not transmitted over the internet, eliminating privacy concerns. Storing a data series using MashableData.com's technology is less traceable than downloaded a Excel or CSV file. update 6/1/12

Key proof of concepts

  1. Sharing data between data visualization of different web pages:
    • Population of California graph
    • Population of Texas graph
  2. Safe cross domain data storage using JavaScript message posts and message listeners

Mashable Data Format v0.1

Data and metadata must be serialized into a single string. Double pipes (ie. ||) separation rows and single pipes (ie. |) separate cell. Carriage return characters and linefeed characters are not allowed. The use of pipe characters is reserved for delimiters. Empty values are not allowed. Use the string null instead:
